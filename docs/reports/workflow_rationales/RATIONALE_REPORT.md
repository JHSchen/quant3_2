# 量化指标选择理由报告 (Rationale for Quantitative Metrics)

## 1. 策略代理指标：为什么选 ROIC 和 FCF？
- **ROIC (资本回报率)**：这是衡量企业护城河最直接的指标。高 ROIC 意味着企业在产业链中拥有极强的议价权（拓扑优势）。
- **FCF (自由现金流)**：利润可以作假，但现金流很难。这是防止“暴雷”的第一道防线。

## 2. 对手代理指标：为什么选 Altman Z-Score 和 F-Score？
- **Altman Z-Score**：这是金融工程中预测破产最经典的模型，准确率高达 80%-90%。对手代理需要这个工具来剥离管理层的“修辞”。
- **Piotroski F-Score**：这是一种非线性打分系统，涵盖了杠杆、流动性、运营效率。它可以识别出那些虽然利润在增长，但资产负债表在腐烂的公司。

## 3. 风险代理指标：为什么选 Beta 分解和凯利公式？
- **Beta 分解**：很多投资者误把行情好当成自己水平高。通过分解 Beta，我们能识别出哪些是市场给的（Beta），哪些是企业竞争优势给的（Alpha）。
- **凯利公式**：这是职业赌徒和顶级对冲基金（如文艺复兴技术）长盛不衰的秘诀。它能根据胜率和赔率算出“最优下注比”，防止单次亏损导致系统性毁灭。

## 4. 理由：为什么需要“审计评分卡” (Audit Scorecard)？
- **反馈闭环**：如果不打分，AI 就会变得模糊。Scorecard 是对分析质量的量化评估，便于我们在 `ADAPTATION_LOG.md` 中进行复盘迭代。
